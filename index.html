Enron Email Dataset Analysis

Created as a final project for MATES Data Science Course 2020-2021

Enron Email Analysis

In the attached notebook I conduct analysis on the infamous Enron Email Dataset, which was originally obtained during investigations following the company's 2001 filing of Chapter 11 Bankruptcy.

Throughout the project I demonstrated my ability to sift through and format complicated datasets, create interesting and useful data visualizations, create network vizualizations, create word clouds, and use machine learning algorithms (specifically Natural Language Processing and K-Means Clustering).

Predicting Severe Coral Reef Bleaching Based on Global Anthropogenic Activity

Created as a final project for MATES Oceanography Course 2021
Research Poster

Placeholder for Github Link

My group mates and I used the experience we obtained in our Data Science class to use machine learning algorithms to predict whether a country will have severe bleaching based on anthropogenic activity.

This project displays understanding of both KNN and ANN algorithms as well as experience with making complex data visualizations, such as the heatmaps that can be found in the linked poster above. The github repository includes our jupyter notebooks, research poster/proposal, and links to the utilized datasets.
